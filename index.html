Vad är Reinforced Learning?

Välkommen!

Syftet med hemsidan är att lära och inspirera andra i att följa inlärningens fotspår.
Jag vill nå dig som kan vill doppa tårna i nytt vatten så att jag kan locka dig att även doppa huvudet.

Denna hemsida kommer visa dig hur du kan börja lära agenter inuti spelmotorn Unity.
Jag vill få dig att själv prova på detta och sedan bygga vidare på vad jag lärt ut.

Instruktionerna är anpassade för att lära dig som nybörjare steg för steg hur du kan göra för att utan problem påbörja inlärning.

Hemsidan är gjord som ett skolarbete och är inte ansedd för kommersciellt bruk.

Om du inte förrän nu har hört talas om Reinforced Machine Learning kan du gärna ta en titt på den bifogade videon skapad av AI Warehouse på youtube, så hoppas jag att den kan fånga ditt intresse.

om det finns något du vill tycka till om angående hemsidan kan du gärna göra det nedanför i Google formuläret. Jag tar gärna emot lite kritik.
